## Установка

1. Склонируйте репозиторий:

<code>git clone</code>

2. Перейдите в директорию приложения:

<code>cd SPA-for-currency-conversion.</code>

3. Установите зависимости:

<code>npm install</code>

## Запуск

Чтобы запустить приложение, используйте команду:

<code>npm start</code>
